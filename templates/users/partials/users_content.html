Información sobre Gestión de Usuarios

Objetivo: En esta sección puedes administrar las cuentas de usuario del sistema, permitiendo gestionar accesos y roles.

Búsqueda: Usa el cuadro de búsqueda para localizar rápidamente usuarios por nombre, correo o rol.

Visualización: La tabla muestra información general como nombre, correo, rol y estado del usuario.

Edición: Puedes editar los datos de un usuario desde las acciones disponibles en la tabla. En esta sección no se crean usuarios; solo se gestionan.

Desactivación: Puedes desactivar temporalmente un usuario sin eliminarlo, evitando su acceso al sistema.

Consejo: Asegura que cada usuario tenga asignado el rol correcto para evitar accesos indebidos.
